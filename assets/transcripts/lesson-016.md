# 科学的説明・反証可能性 台本

今日は「科学哲学・技術哲学」の中から「科学的説明・反証可能性」を扱います。

最初の問いは、科学的な主張と、そう見えるだけの主張はどこで分かれるのか。

ポイントは三つです。

1. 科学的説明の特徴を説明できる
2. 反証可能性の意義と限界を理解できる
3. 仮説検証の基本構造を整理できる

本文では、「科学は説明を作る営みである」、「反証可能性の力」、「反証可能性だけでは足りない」、「再現性と信頼」の順に進みます。

聞きながら、重要語を暗記しようとするより、自分が最近経験した判断や対話に置き換えてください。哲学の概念は、生活の外に飾るものではなく、生活の中の見えにくい構造を見えるようにする道具です。

最後の演習は次の通りです。

気になる科学的主張を一つ選び、どんな観察があれば反証されるか、どんな再現条件が必要かを書く。
