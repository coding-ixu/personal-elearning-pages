# 権力・承認・社会構造 台本

今日は「政治哲学・社会哲学」の中から「権力・承認・社会構造」を扱います。

最初の問いは、個人の努力では説明できない不利益は、どのように社会の中で作られるのか。

ポイントは三つです。

1. 権力の見えにくい働きを説明できる
2. 承認の重要性を理解できる
3. 構造的問題と個人責任を区別できる

本文では、「権力は命令だけではない」、「承認は贅沢ではない」、「構造は個人を免責するだけではない」、「規範を問い直す」の順に進みます。

聞きながら、重要語を暗記しようとするより、自分が最近経験した判断や対話に置き換えてください。哲学の概念は、生活の外に飾るものではなく、生活の中の見えにくい構造を見えるようにする道具です。

最後の演習は次の通りです。

自分の身近な制度や職場ルールを一つ選び、それが誰にとって使いやすく、誰にとって負担が大きいかを分析する。
