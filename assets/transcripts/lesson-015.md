# 自己・他者・コミュニケーション 台本

今日は「心の哲学・言語哲学」の中から「自己・他者・コミュニケーション」を扱います。

最初の問いは、自分の心や他者の心は、どのように理解されるのか。

ポイントは三つです。

1. 自己理解の限界を説明できる
2. 他者の心の問題を理解できる
3. 対話における解釈の役割を考えられる

本文では、「自分の心も透明ではない」、「他者の心の問題」、「解釈は共同作業である」、「共感と境界線」の順に進みます。

聞きながら、重要語を暗記しようとするより、自分が最近経験した判断や対話に置き換えてください。哲学の概念は、生活の外に飾るものではなく、生活の中の見えにくい構造を見えるようにする道具です。

最後の演習は次の通りです。

最近の誤解を一つ選び、共有されていなかった前提、使った言葉の違い、確認すべきだった点を三つに分ける。
