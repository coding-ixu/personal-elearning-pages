# 人生哲学を実践にする 台本

今日は「実存主義・人生哲学」の中から「人生哲学を実践にする」を扱います。

最初の問いは、哲学を読んだ後、生活はどう変わりうるのか。

ポイントは三つです。

1. 哲学的内省を習慣化できる
2. 判断、対話、記録に哲学を使える
3. 自分の問いを育てる計画を作れる

本文では、「哲学は生活の検査道具である」、「小さな問いを持つ」、「対話で考えを鍛える」、「記録して変化を見る」の順に進みます。

聞きながら、重要語を暗記しようとするより、自分が最近経験した判断や対話に置き換えてください。哲学の概念は、生活の外に飾るものではなく、生活の中の見えにくい構造を見えるようにする道具です。

今日から七日間、毎日一つだけ哲学的問いを書き、翌日に一文で答えを更新する。
