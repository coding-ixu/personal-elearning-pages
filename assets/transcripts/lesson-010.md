# 正義・自由・平等 台本

今日は「政治哲学・社会哲学」の中から「正義・自由・平等」を扱います。

最初の問いは、公正な社会とは、何をどのように分配し、何を守る社会なのか。

ポイントは三つです。

1. 分配的正義の問いを説明できる
2. 自由の複数の意味を区別できる
3. 平等と公平の違いを考えられる

本文では、「正義は分配の問いである」、「自由には複数の意味がある」、「平等は同じ扱いだけではない」、「権利は境界線を引く」の順に進みます。

聞きながら、重要語を暗記しようとするより、自分が最近経験した判断や対話に置き換えてください。哲学の概念は、生活の外に飾るものではなく、生活の中の見えにくい構造を見えるようにする道具です。

最後の演習は次の通りです。

教育、医療、住居、表現の自由のいずれかについて、自由・平等・権利の三語で制度評価を書く。
